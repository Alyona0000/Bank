Опис проєкту:
Даний проєкт створений для моделювання роботи банку: депозитів, кредитів, цінних паперів та щоденних фінансових операцій.

Для більшості звичайних користувачів робота через термінал є незручною, тому був розроблений веб-інтерфейс на HTML, CSS та JavaScript.

Веб-інтерфейс дозволяє взаємодіяти з банківською системою у графічному середовищі без необхідності роботи через командний рядок.

Серверна частина написана мовою C++, що дозволяє виконувати математичні обчислення, моделювання фінансових процесів та обробку банківських операцій на сервері.

Як банк рахує?

Котик-фінансист думає над прибутком банку...

Капітал банку за день:
A = A + D - K + KP - DP + SP
D — депозити:
D = сума грошей, які клієнти принесли в банк
K — кредити:
K = сума грошей, які банк видав клієнтам
KP — прибуток з кредитів:
KP = кредити × 18% / 365
DP — витрати на депозити:
DP = депозити × 8% / 365
SP — прибуток з акцій:
SP = кількість проданих акцій × ціна акції
